Flaskе…Ҙй—Ё

еҸ‘иЎЁдәҺ 29-12-2020 жӣҙж–°дәҺ 30-12-2020 еҲҶзұ»дәҺ Notebook йҳ…иҜ»ж¬Ўж•°пјҡ

FlaskжЁЎеқ—еҢ–з®ЎзҗҶ

BluepointеҸҜд»Ҙе®һзҺ°Flaskзҡ„жЁЎеқ—еҢ–з®ЎзҗҶпјҢеңЁжҜҸдёӘеҢ…зҡ„__init__ж–Үд»¶дёӯеҸҜд»Ҙе®ҡд№үеҜ№еә”зҡ„и“қеӣҫгҖӮи“қеӣҫд»…жҳҜйқҷжҖҒзҡ„пјҢйңҖиҰҒеңЁеә”з”ЁиҝҗиЎҢж—¶пјҢеҜ№е…¶иҝӣиЎҢиЈ…иҪҪгҖӮ

жіЁж„Ҹ

  1. __init__ж–Үд»¶дёӯе®ҡд№үеҘҪи“қеӣҫеҗҺдёҖе®ҡиҰҒе°ҶеҢ…еҗ«и·Ҝз”ұзҡ„ж–Үд»¶importиҝӣжқҘпјҢеҗҰеҲҷдёҚиғҪиҜҶеҲ«зӣёеә”зҡ„и·Ҝз”ұгҖӮ
  2. importж–Үд»¶ж—¶дёҚиғҪдҪҝз”ЁйҖҡй…Қз¬Ұ*иҝӣиЎҢеҜје…ҘгҖӮ

Flaskдёӯзҡ„requestдёҺrespond

request

from flask import request

  1. paramдј еҸӮ
    дҪҝз”Ёrequest.args.get(вҖҳargnameвҖҷ)иҺ·еҸ–й”®еҖјеҜ№гҖӮ

  2. bodyдј еҸӮ
    дҪҝз”Ёrequest.form.get(вҖҳargnameвҖҷ)иҺ·еҸ–formж јејҸдј еҸӮй”®еҖјеҜ№гҖӮ
    дҪҝз”Ёrequest.get_data()иҺ·еҸ–е…ЁйғЁrawж јејҸеҶ…е®№гҖӮ
    дҪҝз”Ёrequest.get_json()иҺ·еҸ–е…ЁйғЁrawж јејҸеҶ…е®№пјҢдё”еҝ…йЎ»дёәjsonж јејҸгҖӮ

жіЁж„Ҹ
form-data: еҸҜд»Ҙдј й”®еҖјеҜ№е’Ңж–Үд»¶гҖӮ
x-www-form-urlencodes: еҸӘиғҪдј й”®еҖјеҜ№гҖӮ

respond

from flask import jsoniify
import json

  1. зӣҙжҺҘиҝ”еӣһjson
  • jsonify()еҸҜд»ҘзӣҙжҺҘеҲ¶дҪңjsonиҝ”еӣһ (иҝ”еӣһзҡ„ж јејҸд№ҹе®ҡд№үдёәapplication/json)
  • json.dumps()еҸҜд»Ҙе°Ҷdictз”ҹжҲҗjson (иҝ”еӣһзҡ„ж јејҸй»ҳи®Өдёәtext)
  1. з®Җжҳ“е®ҡд№үmake_respond
args еҗ«д№ү
respond иҝ”еӣһзҡ„еҶ…е®№data
status_code иҝ”еӣһзҠ¶жҖҒз Ғ
headers иҝ”еӣһеӨҙйғЁдҝЎжҒҜдёәй”®еҖјеҜ№
  1. иҜҰз»Ҷе®ҡд№үRespond
args еҗ«д№ү
respond иҝ”еӣһзҡ„еҶ…е®№data
status иҝ”еӣһзҠ¶жҖҒ ж јејҸдёәвҖҷ200 OKвҖҷ
status_code иҝ”еӣһзҠ¶жҖҒз Ғ
headers иҝ”еӣһеӨҙйғЁдҝЎжҒҜдёәй”®еҖјеҜ№
set_cookie и®ҫзҪ®cookie
del_cookie еҲ йҷӨcookie

Flaskдёӯзҡ„session

from flask import session

  1. дҪҝз”ЁsessionйңҖиҰҒи®ҫзҪ®config
    app.config[вҖҳSECRET_KEYвҖҷ]=os.urandom(24)

  2. FlaskжҸҗдҫӣзҡ„sessionжҳҜеӮЁеӯҳеңЁcookieдёӯзҡ„
    session[вҖҳusernameвҖҷ] = data[вҖҳusernameвҖҷ]
    жӯӨж—¶зҡ„cookieдёӯеӯҳеӮЁдәҶеҠ еҜҶиҝҮзҡ„usernameгҖӮ

  3. иҺ·еҸ–session
    session.get(вҖҳkeyвҖҷ)жҲ–session[вҖҳkeyвҖҷ]
    дҪҶжҳҜеҗҺиҖ…еҰӮжһңжІЎжңүиҝҷдёӘkeyдјҡжҠҘй”ҷгҖӮ
    иҺ·еҸ–еҲ°зҡ„sessionиҮӘеҠЁз»ҸиҝҮдәҶи§ЈеҜҶгҖӮ

  4. еҲ йҷӨsession
    session.pop(вҖҳkeyвҖҷ)жҲ–session[вҖҳkeyвҖҷ]=False
    session.clearжё…йҷӨsessionдёӯжүҖжңүж•°жҚ®гҖӮ
